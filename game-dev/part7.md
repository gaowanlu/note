# 🚗 MYSQL 数据库

## MYSQL 数据库

存储是网络游戏开发中不可缺少的部分。在存储方面，不同的游戏公司采用不同的数据库，有使用 MySQL 的，也有使用 MongoDB 的。游戏数据库版本迭代加快，直接在数据库中存储 Blob 数据，
是与 Web 较为不同有特色的点。
